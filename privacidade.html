Aviso de Privacidade

Portal Controle de Compras Disktrans — T.I.

Quais dados este portal trata

Para operar o controle de compras de equipamentos de T.I., este portal trata os seguintes dados pessoais:

Dados de conta: nome e e-mail, usados para autenticação (login) e para identificar quem criou um pedido ou tomou uma decisão de aprovação.

Dados de pedidos de compra: descrição do item, fornecedor, valor, link de compra, departamento e um campo de texto livre de solicitante, associados a quem criou o pedido.

Dados de auditoria: histórico de aprovações e reprovações, incluindo quem tomou a decisão, quando e uma observação opcional — mantido de forma imutável para fins de auditoria interna.

Para que esses dados são usados

Autenticação e controle de acesso ao portal, conforme o papel de cada pessoa (solicitante, aprovador ou administrador).

Registro, aprovação e acompanhamento de pedidos de compra de equipamentos de T.I. da Disktrans.

Auditoria interna: manter um histórico confiável de quem aprovou ou reprovou cada pedido.

Geração de relatórios gerenciais agregados (por mês, fornecedor, departamento) para apoiar decisões internas.

Onde os dados ficam armazenados

Os dados são armazenados no Supabase (banco de dados na nuvem), protegidos por controle de acesso por linha (Row Level Security): cada pessoa só acessa os dados que sua função autoriza. O acesso é restrito a colaboradores autorizados da Disktrans — os dados não são vendidos, compartilhados ou usados para fins de publicidade.

Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei 13.709/2018), você pode solicitar, a qualquer momento, confirmação de que seus dados são tratados, acesso aos dados, correção de dados incompletos ou desatualizados, e informações sobre com quem os dados são compartilhados.

Para exercer esses direitos, entre em contato com o responsável pelo tratamento de dados na Disktrans [e-mail/contato do responsável a ser definido pela empresa].
